What is the main goal of this website?

The main objective is to allow you to extract data from the bycicle tripdata database through a series of customizable SQL queries, and obtain beautifully plotted data to obtain useful insights.

How the idea for this project originated?

This project has been developed as part of CS50's final project.

The main idea for this website came to me as I was working on a data science project. The idea of letting the user pick what variables they wanted to analize and provide them with a visual representation of the data sounded like a useful and helpful solution.

The decision to select the data came from the Google Analytics Professional Certificate featured in Coursera, although as mentioned in the data webpage, the data comes from Devvy, this dataset was featured in one of Google Certificate's courses and the familiarity was definitely a plus and it made it an easy decision.
